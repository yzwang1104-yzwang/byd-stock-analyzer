# PITFALLS.md — 领域陷阱

**Domain:** 股票分析信号工具
**Confidence:** MEDIUM

---

## 关键陷阱（会引发重写/法律风险/信任摧毁）

### #1 前瞻偏差（Look-Ahead Bias）— 最高技术风险
指标计算意外使用了未来数据。回测完美，实盘失败。
**预防：** 始终 `df['close'].shift(1).rolling(20).mean()` 而非 `df['close'].rolling(20).mean()`；编写"无前瞻"测试。
**Phase:** 1 — 第一天就要正确。

### #2 AkShare 数据质量
免费社区 API 可能返回未复权价格（除权除息后看起来像暴跌）、停牌期间返回最后成交价、数据陈旧无标记。
**预防：** 始终用 `qfq`（前复权）选项；数据验证检查（假期/异常跳变）；CSV 激进缓存。
**Phase:** 1

### #3 评分的虚假精度
输出"78.5/100"或"仓位 32.7%"，用户视为科学确定性——但其实只是几个粗糙指标的加权。
**预防：** 整数评分；置信带；明确告知"误差约 ±10 分"。
**Phase:** 1

### #4 重复信号——"狼来了"问题
下跌趋势中估值指标越来越"便宜"，连续发出买入信号，用户持续加仓后大幅亏损，永远不再信任工具。
**预防：** 趋势动量纳入评分；信号冷却期；追踪信号准确率并透明展示。
**Phase:** 1（评分模型）+ 2（历史追踪）

### #5 投资顾问法律风险（中国）
Phase 3 向付费用户发放买卖信号几乎肯定构成证券投资顾问业务，需要 CSRC 牌照（《证券投资顾问业务暂行规定》）。
**预防：** Phase 1/2 仅限个人使用；所有输出加免责声明；Phase 3 前咨询证券律师。
**Phase:** 1（免责声明嵌入）→ 3（法律审查必须）

---

## 中等陷阱

### #6 评分模型确认偏差
模型被校准为确认创始人已有信念（BYD 被低估），80% 的时间输出"买入"。
**预防：** 必须包含看跌指标（RSI 超买等）；回测至少 3 种场景；如 >60% 天数为买入信号则重新校准。
**Phase:** 1

### #7 比亚迪特定幸存者偏差
模型针对比亚迪历史模式调优，政策/竞争格局变化后失效。
**预防：** 区分"通用原则参数"与"比亚迪特定参数"；跨子周期测试（2015-18 / 2020-21 / 2022-24）。
**Phase:** 1 + 持续

### #8 信号输出信息过载
开发者展示所有工作（PE、MACD、RSI、成交量……），用户回到原点——淹没在数据中无法决策。
**预防：** 主输出 = 一行（分数 + 操作 + 仓位）；详情折叠隐藏；不超过 3 个数字值。
**Phase:** 1（CLI 输出格式）

### #9 缺失卖出信号
用户买在好价格，持有过顶后回落，最终收益平庸。工具"成功"但用户未兑现。
**预防：** Phase 1 至少加基础卖出触发（RSI > 70 + PE > 80 分位）。
**Phase:** 1 — 必须解决

## 次要陷阱

### #10 AkShare 单点故障
API 变更、故障或被墙 → 工具彻底停摆。
**预防：** 抽象数据层接口；至少一个备用数据源；CSV 缓存+预警。
**Phase:** 1（抽象层）→ 2（缓存+备用）→ 3（考虑付费数据）

### #11 忽略宏观和政策因素
只分析价格和基本面，忽略利率/补贴/关税/大宗商品。
**预防：** Phase 1 可接受；Phase 2+ 添加"宏观风险折扣"手动输入因子。
**Phase:** 2+

### #12 CLI→Web 架构重写风险
Phase 1 写成单片脚本，迁移到 Django 时全部重写。
**预防：** 从第一天起分离三层（数据获取/分析逻辑/输出格式化）。
**Phase:** 1

### #13 验证前过度工程
在用户验证基础评分模型有用之前就加 ML/LSTM/情感分析。
**预防：** Phase 1 严格锁定简单指标；设置"复杂度预算"。
**Phase:** 1

### #14 单股票风险
工具实用性完全绑定比亚迪叙事——长期下跌时用户停止使用。
**预防：** 接受为已知限制；下跌期间聚焦"何时退出/重入"。
**Phase:** 3 重新评估

### #15 用户不执行信号（元风险）
工具有效、信号准确——但用户仍然犹豫。根本原因不是信息，是心理。
**预防：** 输出设计制造紧迫感（"信号 24 小时内有效"）；追踪"信号给出 vs 实际执行"；显示不作为的代价。
**Phase:** 1（行动导向输出设计）→ 全周期
